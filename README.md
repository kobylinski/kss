# Install
```
$ npm install niepsuj-kss-template
```

# Use

```
$ kss \
    --source "source folder" \
    --destination "destination folder" \ 
    --builder "path to template" \
    --custom "custom html - footer logo" \
    --js "list/of,js/files,comma/separated" \
    --css "list/of,css/files,comma/separated"
```
